# 🌍 LexiCore AI

A powerful, high-performance **bi-directional linguistic translation and audio synthesis framework** built using **Streamlit**, **Deep Translator**, and **gTTS (Google Text-to-Speech)**. This app enables users to translate text between languages and instantly hear the pronunciation with a clean, modern interface.

---

## 🚀 Features

* 🌐 Bi-Directional Translation: Translate seamlessly between languages in both directions.
* 🔊 Real-time Audio Synthesis: Integrated Text-to-Speech (TTS) for both source and target text.
* 🔄 Swap Translation: Easily switch between source and target languages.
* 🎨 Modern UI: Glassmorphism-inspired design with responsive layout.
* 📋 Clean and professional interface.

---

## 🛠️ Tech Stack

* **Frontend & Backend:** Streamlit
* **Language:** Python
* **Translation API:** Deep Translator
* **Audio Engine:** gTTS (Google Text-to-Speech)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone [https://github.com/akshay-saxena1/LexiCore-AI-language-translator-app](https://github.com/akshay-saxena1/LexiCore-AI-language-translator-app)
cd LexiCore-AI-language-translator-app
```

2.Create and activate a virtual environment (optional but recommended):

```bash
# Create the environment
python -m venv venv

# Activate on Windows
venv\Scripts\activate

# Activate on macOS/Linux
source venv/bin/activate
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```


4. Run the app:

```bash
streamlit run app.py
```

## 💡 How to Use 

1.Enter Text: Type or paste your content into the Source input box.

2.Configure: Select your desired Source and Target languages from the dropdowns.

3.Translate: Click the **Translate** button to process the text.

4.Listen: Click the Audio buttons to hear the real-time text-to-speech synthesis for both input and translated text.

5.Manage: Copy the output or download the results as needed.
