### 📖 About the Project
LexiCore AI is a high-performance, web-based translation application. Designed with modern aesthetics and powered by robust translation engines, it bridges linguistic gaps by providing seamless, bi-directional text translation and real-time audio synthesis.

### ✨ Key Features
* **🌍 Multi-Language Support:** Translate between English, Hindi, Marathi, Gujarati, Bengali, Telugu, Tamil, and Kannada.
* **🔊 Real-time TTS:** Integrated Google Text-to-Speech (gTTS) to hear the translation instantly.
* **🎨 Sleek UI:** Glassmorphism-inspired design with a dynamic breathing gradient background.
* **⚡ Fast & Efficient:** Built using Streamlit for rapid, responsive performance.
